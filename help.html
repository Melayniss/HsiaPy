探索之源

当你看到PythonWebEditor（在线编译器）时，你一定会很困惑该如何使用吧？不必烦恼！就让我们一起来看看Python在线编辑器有哪些部分，并且具备了何种功能吧！

首先，你需要了解在编译器顶部显示的下列按钮，所代表的含义：

按下TAB键会将焦点移动到按钮之间。按ENTER将激活高亮显示的按钮。如果你在文本窗口中，你将可能需要按ESCAPE键（Esc键）来重新聚焦在按钮上。当然，你也可以使用鼠标来进行操作。

这些按键是用来干什么的？

单击“云端上传”按钮，将您的代码保存到云服务器上。 之后服务器会返回显示您的保存结果。 若保存成功，则您可以通过点击“云端下载”按键来查看所有您保存的文件。 若保存失败，则服务器会返回保存失败的原因。 您可以通过检查对应原因，进行相应的操作，再重新尝试上传。
单击“云端下载”按钮，浏览器将自动跳转到您在云端服务器上保存的所有文件页面。 您可以在该页面查看所有您保存的文件。 您也可以点击任一平台下的“新建项目”，来跳转至对应的平台进行新建。 点击任意一个您保存的文件，您可以在对应的平台上进行编辑，并进行相应的操作。
从你的电脑很容易就能加载文件到编辑器: 单击“打开按钮，然后将文件从计算机拖放到“打开”区域。 或者，您可以使用该文件选择器(只需点击底部的链接以切换两个选项)。 编辑器知道如何读取Python文件(文件名以“.py”结尾) 并从hex文件中提取Python代码(文件名以“.hex”结尾)。
单击“下载”按钮，将代码以“.hex”为后缀的文件格式保存到您的计算机。 之后您可以接入您的硬件设备(它会显示为USB存储），并将新保存的文件拖放到设备上。 如果您编写的代码无误，它将在设备上开始运行。 若出现错误时，设备的显示屏上将滚动显示一条错误消息。 想知道如何从一个hex文件中提取代码到编辑器中，请参阅下一按钮的介绍。
单击“保存”按钮，将代码以“.py”为后缀的文件格式保存到您的计算机上。 因为您正在下载Python代码，您的计算机可能会询问您是否想要保存文件。 它是在预防您从互联网下载恶意软件。 但由于这是您自己的代码，您可以安全地继续。 想了解如何从python文件中加载代码返回到编辑器，请参阅下一按钮的介绍。
这是一个实验特性。它可能在未来发生变化并可能包含错误。 点击“模块化”按钮，将会弹出一块弹窗。 弹窗中将显示一列导航，每一个标签中将会有一组拖放。 您可以通过拖放至面板上，用来快速创建Python代码。 通过移动、拼接拼图模块，来观察Python代码的出现和更改。 移动这些块将重写Python代码，您可能会丢失工作。 此外，块的排列目前没有保存，只是产生的Python代码。
单击“代码框架块”按键，将会弹出一个Python“代码片段”菜单。 代码片段是可在您自己的程序中重用的短代码块。 对于您想要使用Python做的最常见的事情，有一些代码片段。 选择您想要的，并填充代码编辑器中的空白，使其按照您想要的方式工作。 当启用代码拼图时，框架块将被禁用
点击“帮助”按钮就可以到达这里。 但您可能已经知道了，对吧?___:)___ 注意，您的编辑器仍然可用。 它在您的浏览器中有一个单独的选项卡。
单击“分享”按键，创建一条唯一的链接来指向你的脚本。 将此链接添加到您的书签，以便检索代码。 如果你分享这个链接，其他人就能看到你的代码。 因此，您的代码仍然是私有的，您必须为它设置一个密码和一个可选的密码提示。 如果从本地文件系统运行编辑器，则此按钮不可用。
每个人都喜欢炫耀他们出色的Python技能。 当您试图通过投影仪向一大群人展示代码时， 这两个按钮对于放大和缩小特别有用。

脚本的名称和描述显示在右上角。单击，可进行编辑。

编辑器的首页部分是文本编写窗口：

使用起来十分简单，只需要点击和编辑即可。

编辑器试图通过给文本着色来帮助您了解程序的所有不同部分。 例如，Python关键字(内置于Python语言中的词)是灰色的。 颜色较亮的词是 您已经创建的程序的一部分。 棕色的单词是不变的值，紫色的单词表示要显示的字符串。 所有行都用高亮显示的当前行编号。

想知道如何在显示屏上滚动显示一条消息吗？ 您只需要按真实的Python代码来编写即可。 本编辑器中的拼图块及框架代码语句，基本都是为了在小型设备上烧录而准备的。 以下是一个在您的小型设备显示器上滚动显示消息所需的代码。 复制下来并粘贴在编辑器中，下载它并复制到您的设备当中 （可参考“本地保存”按键的介绍）。

from microbit import *


	display.scroll("Hello, World!")

第一行 (from microbit import *) 是告诉Python在线编译器获取所有的Python代码块，以便编写。

第二行(display.scroll("Hello, World!")) 是告诉Python编辑器使用其中的一个片段 (the display module) 来滚动文本，"Hello, World!" 通过设备前面的物理显示器，您可以清晰的看到语句的滚动显示：

代码拼图

通过拖放和组装包含代码的模块化拼图，来创建新的Python代码

代码拼图会在您拖动模块时在文本编辑器中实时更新您的Python代码。 Python代码拼图中有许多不同的模块，来覆盖您设备所需的所有特性。 通过将不同的代码模块组合在一起(如上面的示例)，您可以生成更高级的代码。

您可以使用代码拼图来创建 新代码，且因为拖动模块会重写您在编辑器中的任何已有代码 （并且您可能会因此而丢失当前工作数据）。 块的排列方式(目前)还没有保存，因此“代码拼图”可以很好地制作简单的Python代码，无需大量输入。 最后，您将了解到，要想简单地输入Python，这是迄今为止编写代码最快、最简单的方法。

通过“代码拼图”并不能获得Python编辑器的所有特性。 一些更高级、更有趣的内容只在纯Python中可用。

但是，通过使用“代码拼图”，您将很快掌握Python代码是如何工作和组合在一起的。

BUG 调试

Things will go wrong!

You have to imagine Python is the most strict English teacher in the universe... yes, even more strict than that really strict one you have at your school. Put simply, you have to type Python without any mistakes for it to work.

All programmers make mistakes and create bugs. It's a fact of life. When you have a bug MicroPython tries to help you out: it will scroll a message on its display. It may even include a line number. This message is like the first clue in a treasure hunt, you have to figure out the rest yourself.

Common bugs include Syntax Errors (which means you've typed it in wrong) and Name Errors (that mean you've typed in correct Python, but it can't work out what you're coding about).
